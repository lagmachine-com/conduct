<h1 align="center">Conduct</h1>
<p align="center">Generic asset and version management for creative projects</p>

Conduct provides a highly configurable pipeline and version management system for creative projects, enabling strict but easy handovers of work across departments, by defining processes in configuration and code.

> [!NOTE]
> Conduct is still a work in progress, and the api is *not* yet considered stable. 


## Integrations
Also contained in this repo are plugins to integrate Conduct in to third party software. Conduct provides a command line api, which has also been designed to be called programatically. These plugins typically exist mainly as a UI which interfaces with conduct via commands, in order to reduce the amount of code which needs to be re-written per integration and provide a more consistent user experience. 

### Currently Supported:
- Blender
- Inkscape
